# 🚌 RedBus Clone Project

## Features
- User Login & Signup
- Bus Search
- Seat Booking System
- Payment Flow
- Booking History

## Tech Stack
- Java (Spring Boot)
- MySQL
- Thymeleaf
- Bootstrap

## How to Run
1. Import project in Eclipse
2. Setup MySQL using database.sql - sample routes data (buses)
3. Update application.properties - set db-username , db-password
4. Run Spring Boot
